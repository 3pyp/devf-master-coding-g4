Ejercicios Express:

Enunciados

  • Crea un backend con express y agrega un endpoint '/api/v1/suma' que responda a una petición de tipo POST con la suma de dos números que reciba mediante el body como num1 y num2. El servidor debe responder con un mensaje con el resultado de la suma: EJ: 'El resultado es 7'
  • Agrega un endpoint '/api/usuario/' que responda a una petición de tipo GET con el nombre que sea recibido a través de params. El servidor debe responder con un código de estado 200 y un json como este: { 'usuario': 'Pepito' }
  • Crea un endpoint para una petición de tipo DELETE donde envíes un ID (un número cualquiera) a través de params. Si el param contiene el ID 3, entonces responde con un status 200 y el mensaje "se ha eliminado el objeto con ID 3", de lo contrario, si envían cualquier otro número como ID, responde con un status 404 y el mensaje "No se encontró el objeto con el ID especificado".
